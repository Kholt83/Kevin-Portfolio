Project Overview

FANUC MT-Linki Machine Monitoring Implementation Collaborated on the successful deployment of FANUC MT-Linki across three CNC machines to enable real-time machine monitoring, alarm tracking, and uptime analytics. Took lead on network and machine-side integration, including configuring Ethernet communication via FOCAS2, assigning static IPs, and updating machine options to support MT-Linki connectivity.

Migrated the entire CNC machine network to a new IP family to resolve firewall and subnet conflicts, ensuring all equipment could reliably communicate with the MT-Linki server. Led the SCADA integration effort by collecting machine-side data and building out license configurations, achieving 70–80% of the integration scope. Developed uptime reporting workflows that funneled data into SmartSheets for cross-team visibility and executive dashboards.

The project involved close collaboration with the IT team to ensure secure and reliable network connectivity, as well as coordination with machine operators to validate data accuracy and usability. Key challenges included troubleshooting initial communication issues, optimizing data collection intervals, and ensuring compliance with cybersecurity protocols.
