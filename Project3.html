Michelle Nails & Spa — Business Website Deployment

ROLE:

Freelance Full-Stack Developer, Systems Admin

TOOLS:

VS Code, HTML, CSS, Ubuntu Server, Nginx, Cloudflare, Git, GitHub

Project Overview

Built and deployed a full-stack business website for Michelle Nails & Spa, a local beauty and wellness business. This project included not only frontend development but also infrastructure deployment and migration from legacy hardware.

Key Highlights:
  • Created a responsive, branded website with relaxing visuals, spa-oriented color palette, and intuitive layout.
  • Configured Ubuntu Server with Nginx to host the site on a self-managed homelab environment.
  • Integrated Cloudflare for DNS and SSL, enabling secure HTTPS traffic to a self-hosted production server..
  • Migrated hosting from Synology NAS to modern Linux stack, reducing latency and improving reliability.
  • Ensured fast load times, mobile accessibility, and SEO metadata to support local business growth.


Outcome:
  • A live, real-world business site that has improved client engagement and professional presence — all managed from a custom-built homelab server.
