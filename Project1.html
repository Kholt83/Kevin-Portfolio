ROLE:

Front-End Developer, Conversational Flow Designer,Back-End/API Developer

TOOLS:

JavaScript (ES6+), HTML5, CSS3, Node.js, Express, JSON knowledge base (RAG-lite), REST API, NGINX, Cloudflare (DNS/SSL), VS Code, Browser DevTools

Project Overview

A full-stack chat assistant with a browser UI and a Node/Express backend that routes intents, serves curated answers from a JSON knowledge base, and enforces guardrails.

Key Highlights:
  • Designed the chat UI and message loop (send/receive, loading states, error handling).
  • Configured Ubuntu Server with Nginx to host the site on a self-managed homelab environment.
  • Implemented intent routing (hours, services, booking info) with a small JSON KB + fallback responses.
  • Added guardrails (no auto-calls, no commitments) and simple rate limiting.
  • Deployed behind NGINX with HTTPS via Cloudflare; logs for monitoring and debugging.
  • Integrated cleanly on the site with a lightweight embed and responsive styling.


Outcome:
  • Instant, consistent answers to FAQs that reduced repetitive inquiries while keeping control on our server.
