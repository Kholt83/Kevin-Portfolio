Portfolio Website Creation (with AI Collaboration)

ROLE:

Project Lead, Content Strategist, Developer

TOOLS:

VS Code, HTML, CSS, JavaScript, TailwindCSS, ChatGPT, Git, GitHub Pages

Project Overview

Built a professional portfolio website from scratch using VS Code, combining classic web development techniques with real-time AI collaboration. The project showcases technical capabilities, personal branding, and modern development workflow integration.

Key Highlights:
  • Ideation: Leveraged ChatGPT for brainstorming, content drafting, and iterative feedback on design and structure.
  • Code-Centric Development: Used VS Code as the primary development environment for efficient code editing, previewing, and Git integration.
  • Responsive & Accessible: Implemented TailwindCSS for rapid styling and mobile-first design principles to ensure a seamless experience across devices.
  • Version Controlled: Maintained with Git and GitHub, allowing for continuous updates, iteration tracking, and collaboration.
  • Privacy-Aware Thinking: Planned for ethical analytics tools like Plausible while staying budget-conscious.


Outcome:
  • A sleek, professional website that not only demonstrates front-end skills and AI-assisted workflows, but serves as a living document of ongoing personal and professional growth.
